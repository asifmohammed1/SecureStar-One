# Security Policy

## Supported Versions

Only the latest release of **SecureStar One** is supported for security updates. 

| Version | Supported |
| ------- | --------- |
| 1.0.x   | ✅        |
| < 1.0   | ❌        |

## Reporting a Vulnerability

We take the security of our Smart Stadium infrastructure projects seriously. If you believe you have found a security vulnerability in SecureStar One, please report it to us by closing an issue or contacting the project lead (Asif).

**Do not report security vulnerabilities via public GitHub issues.**

## AI Ethics & Safety

SecureStar One utilizes **Nvidia NIM** models for security auditing. Our models are tuned to avoid generating malicious code and are strictly focused on defensive remediation and OWASP compliance auditing.
